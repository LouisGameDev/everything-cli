# IPC

Inter Process Communication (IPC) allows applications to communicate with Everything under Windows.

The Everything IPC allows applications to query and get search results from the "*Everything*" database.

You can only communicate with Everything from the local machine.

Use [ETP](/support/everything/etp) if you would like to communicate with Everything over a network.
